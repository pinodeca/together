# Decision Records

Use this directory to track important project decisions.

## Why
Decision records make reasoning auditable for humans and AI agents.

## Naming
- Use incremental file names: `0001-short-title.md`, `0002-next-title.md`.
- Keep titles specific and stable.

## Required Sections
- Context
- Options Considered
- Decision
- Consequences

## Process
1. Propose a decision record in a PR.
2. Collect comments and revisions.
3. Mark status as `accepted` when agreed.
4. Link related records and docs.
