# Code of Conduct

## Purpose
Set clear expectations for respectful and inclusive participation.

## Expected Behavior
- Treat people with dignity and respect.
- Critique ideas, not individuals.
- Avoid harassment, discrimination, and exclusionary language.

## Reporting and Response
- Provide a clear reporting path.
- Ensure impartial review.
- Document outcomes while protecting privacy as appropriate.

## Discord Community Moderation (Stub)
This section defines baseline expectations for moderating the Together Discord server.

### Baseline Requirements
- Align moderation setup and behavior with official Discord guidance:
	- https://support.discord.com/hc/en-us/articles/14078261239831-Clarifying-Server-Types
	- https://support.discord.com/hc/en-us/articles/360035969312-Community-Server-Guidelines
- Apply the same dignity, inclusion, and anti-harassment standards from this code of conduct.
- Keep moderation actions consistent, documented, and reviewable by project maintainers.

### Moderator Onboarding Requirement
- Anyone taking a moderation role must read and follow the official Discord guidance above.
- Moderator onboarding should include clear escalation paths and incident handling expectations.
