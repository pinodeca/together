# Code of Conduct

## Purpose
Set clear expectations for respectful and inclusive participation.

## Expected Behavior
- Treat people with dignity and respect.
- Critique ideas, not individuals.
- Avoid harassment, discrimination, and exclusionary language.

## Reporting and Response
- Provide a clear reporting path.
- Ensure impartial review.
- Document outcomes while protecting privacy as appropriate.
